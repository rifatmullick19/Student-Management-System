# 📘 Student Management System (C++)

## 1. Introduction

This project is a simple Student Management System developed using C++. It demonstrates basic Object-Oriented Programming (OOP) concepts such as class, objects, and member functions. The system stores and displays student details like roll number, name, and marks.

---

## 2. Objectives of the Project

* To understand the concept of classes and objects in C++
* To implement a simple real-life application using OOP
* To store and display student data efficiently
* To improve basic programming and logic-building skills

---

## 3. Methodology / System Design

* A class named **Student** is created
* Data members: roll number, name, and marks
* A member function `display()` is used to show student details
* Three student objects are created in the main function
* Data is assigned manually and displayed using function calls

---

## 4. Implementation / Source Code

The project is implemented using C++ programming language.
It uses:

* Class and object concept
* Member function for displaying data
* Simple structure for easy understanding

The code creates student objects, assigns values, and displays the information on the screen.

---

## 5. Conclusion

This project helped in understanding the basics of Object-Oriented Programming in C++. It demonstrates how classes and objects can be used to manage and organize data. It is a simple and beginner-friendly project useful for learning fundamental programming concepts.
